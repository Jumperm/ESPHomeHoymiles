# ESPHome for Hoymiles HM300
This ESPHome Project use an ESP32 with an NRF24L01+ to get the values from an Hoymiles HM Inverter into Homeassistant. This is a light weight methode beside AhoyDTU and OpenDTU over that i use bevor.

this is a fork from e-2-nomy channel: https://www.youtube.com/c/e2nomy/
